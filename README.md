# React Static Story Page

This project is a simple React application that displays a static story page using reusable components.

## Story
The Tortoise and the Hare

## Components
- Header: Displays the story title
- Content: Displays the story summary
- Footer: Displays the moral of the story

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

## Technologies Used
- React
- Git & GitHub
